# Asistente GUI de Escritura para Typertask (typertask_gui)

Con typertask_gui, ya no tenés que escribir manualmente cada línea de código. Simplemente hacé clic en los botones correspondientes a las acciones que querés realizar y el asistente genera automáticamente el código necesario. ¡Es tan fácil como eso!
